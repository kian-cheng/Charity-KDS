# Charity-KDS
A simple vibe-coded kitchen display system for a charity event

How it works is that there will be a device at the register, and there will be tablets at each station. This system works when all devices are on the same network. 

To setup host/laptop: <br/>
Run node server.js

On host: (if using an iPad as a register, just replace localhost with the IP of the host) <br/>
localhost:3000 (The Register)
